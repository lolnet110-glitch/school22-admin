# School22 Mobile Admin WebView v2

Мобильная админка для будущей упаковки в APK через WebView.

## Добавлено

- главные кнопки на стартовом экране;
- отдельная кнопка добавления ученика;
- список всех учеников;
- добавление категории;
- добавление подкатегории;
- просмотр категорий;
- выбор класса при добавлении ученика;
- выставление баллов ученику.
