# School22 Admin Class Rating v4

Админка под новую логику без ФИО учеников.

## Что изменилось

- Главная кнопка "Управление рейтингом" теперь открывает админку.
- Убрана логика учеников и ФИО.
- Управление классом:
  - количество учеников;
  - баллы класса по категориям;
  - школьная форма.
- Школьная форма:
  - дата проверки;
  - сколько учеников без формы;
  - автоматический расчёт баллов;
  - история проверок;
  - средний балл по всем проверкам.
